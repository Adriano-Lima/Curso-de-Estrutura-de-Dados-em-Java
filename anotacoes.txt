	Pilha 
		LIFO (last in first out)
		metodos:
			top(pega a ref do topo),pop (tira o no topo),push (add no)

	Fila 
		FIFO (first in, first out)
		enqueue (add no), dequeue (tirar no), isEmpty

	Lista encadeada
		lista aponta para primeiro nó, cada nó aponta para o proximo
		metodos: add(), remove(), get(), isEmpty()

	Lista duplamente encadeada
		a lista aponta para o primeiro e ultimo nós
		cada nó aponta para o anteior e proximo nó
	
	Lista circular
		primeiro no chamado de cabeça ultimo de calda
		lista aponta para a calda
